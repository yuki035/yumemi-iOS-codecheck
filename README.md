## 開発概要

### 環境

- IDE: Xcode 15.2
- Swift: 5.9
- 開発ターゲット: iOS17.2

### 構成

- UIの実装: SwiftUI
- アーキテクチャ: MVVM
- ブランチモデル: GitHub Flow

### パッケージ管理

- Swift Package Managerを使用
  - SwiftLintを初めて導入
